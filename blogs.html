Question-2: What is the difference between innerHTML, innerText, and textContent ?

Ans: innerHTML Gets or sets the HTML content inside an element and innerText Gets or sets only the visible text ignores hidden text and scripts and textContent Gets or sets all text content, including hidden text .

Question-3: What is event delegation in the DOM?

Ans: Event delegation is a technique where you add an event listener to a parent element instead of adding it to multiple child elements.

Question-4: What is event bubbling in the DOM?

Ans: Event bubbling is a behavior in the DOM where an event starts from the target element and then propagates (moves up) through its parent elements until it reaches the root .
